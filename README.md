# LightQtDDoSer
Very lightweight crossplatform ddos. Work only on TCP.

## Download binary release: https://github.com/vakarianplay/LightQtDDoSer/releases

***

Targets can load manualy or from text file.

Text file sample:
``` 
example.site:80

example2.site2:443

site.domain:8080

url.local:443 
```
